# Camera Testing Instructions

## Step-by-Step Testing Guide

### 1. Open the Website
- Go to: https://exam-portal-2004.web.app
- Press Ctrl + Shift + R to clear cache

### 2. Login as Student
- Username: (your student username)
- Password: (your student password)

### 3. Start an Exam
- Click "Start" on either:
  - Theory Exam, OR
  - Practical Exam

### 4. Look for the Camera Section
- On the RIGHT side of the screen
- You should see a GRAY sidebar
- Inside it, there's a BLACK box
- Text says "Camera Off"
- Blue button says "Start Camera"

### 5. Click "Start Camera"
- Browser will show a popup asking for camera permission
- Click "Allow" or "Yes"

### 6. What Should Happen
✅ Camera video appears in the black box
✅ You see yourself (mirrored/flipped)
✅ Red "REC" indicator appears in top-right corner
✅ Red dot is blinking/pulsing

## If Camera Doesn't Work

### Check Browser Permissions
1. Click the camera icon in browser address bar
2. Make sure camera is set to "Allow"
3. Reload the page

### Check if Camera is Available
1. Close other apps that might use camera (Zoom, Teams, Skype)
2. Try the test page: open camera-test.html
3. If test page works but exam doesn't, let me know

### Try Different Browser
- Use Google Chrome (recommended)
- Or Microsoft Edge
- Firefox and Safari may have issues
